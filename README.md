## Getting Started

### Installation
#### Locally

To get a local copy up and running follow these simple example steps.

1. Clone the repo
   ```sh
   git clone https://github.com/shin202/movevee-api.git
   cd movies-api
   npm install
   ```
   
2. Start the server
    ```sh
    npm start
    ```

3. Enjoy üòÑ
